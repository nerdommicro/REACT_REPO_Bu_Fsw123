    Complete Week 5 Project #2: Giphy Search

    • Due Sunday by 11:59pm
    • Points 60
    • Submitting a website url or a file upload

    For this assignment, you will create a Giphy search that renders one animated GIF.

    Getting Started

    Complete the following:

    • Create an account on Giphy Developers: https://developers.giphy.com/ (Links to an external site.)
      1. Create an account, then go to the API Quickstart Guide (Links to an external site.)
      2. In your Dashboard (Links to an external site.), click Create an App.
      3. You will then create an API (not SDK), then Next.
      4. Enter a name and brief description of app.
      5. You will receive your API KEY (beta):

       

    • Read through the GIPHY documentation (Links to an external site.).
      • You will use the Search Endpoint (Links to an external site.) + API key as the URL for your fetch request 
      • Use encodeURIComponent (Links to an external site.) for the passed query input by the user.
    • Build a search form with one input and button
      • Will initially render an animated GIF (use a default value as an initial query).
      • User will enter a search term and click the button to render a new image
      • Each new query from the user will dynamically populate the src attribute of an IMG.

          

      • Also dynamically populate the alt value of the image from the GIPHY API.


         
    • Bonus
      • Render a "loading..." message until the new image is displayed.
      • Render a "No Results Found" message if no image is returned.

    • Your app will look something like this:

      View after user clicks button with entered "baseball" as word in input field:

    • Develop and test your React application with the  npm start (Links to an external site.)  script .

    Submission Instructions

    Submission:

    You will submit a screenshot of the rendered output, and the GitHub URL to your assignment files, so I can review your code.

    How to Submit Your GitHub URL :

    1. When youre done, stage, commit, and then push your changes to your Github repository. (Note: Make sure these files are located inside the week's directory inside your FSW-123 Assignments folder on your local machine. If done correctly, your new folder containing this exercise will now show up in your GitHub repository).
    2. Then, copy the URL. Your URL should look similar to this: https://github.com/username/class/week/project
    3. Go back to this assignment in Canvas, click the Submit Assignment button. At the bottom of the page, paste the URL into the Website URL field, and click Submit Assignment.

    How to Take a Screenshot:

    1. Take a screenshot of the required material, ensuring everything can be seen . You can save your screenshot as an image or paste it to a Word document. 
    2. Go back to the assignment in Canvas, click the Submit Assignment button. At the bottom of the page, upload your file.

    Once both  your URL and screenshot have been uploaded, click the Submit Assignment button.
